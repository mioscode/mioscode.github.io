# (mac) terminal 명령어

|명령어|내용|옵션, 기타|
|-|-|-|
|`man <명령어>`|명령어 도움말 보기|종료->q|
|`cd <옵션>`|디렉토리 변경 (change directory) (window: `dir`)|`cd ~` 또는 `cd ~/` 사용자의 홈 디렉토리로 이동<br>`cd ~<userid>` userid의 홈 디렉토리로 이동<br>`cd -` 바로 앞에 작업한 디렉토리로 이동<br>`cd <디렉토리명>` 지정한 directory로 이동<br>`cd /` 최상위 root directory로 이동<br>`cd ..` 현재 작업 directory의 parent directory로 이동|
|`ls <옵션>`|현재 경로 하위 파일/디렉토리 나열 (list of directory) <br>|`-a` 디렉토리 내의 모든파일(.으로 시작되는 파일 포함)을 출력<br>`-i` 파일의 아이노드(inode)번호를 출력<br>`-h` 파일의 크기를 쉬운 단위로 표시(k: 킬로바이트, m: 메가바이트)<br>`-l` 파일의 다양한 정보를 함께 출력(소유자 정보, 권한 정보, 크기, 날짜)<br>`-m` 파일을 쉼표로 구분하여 가로로 출력<br>`-s` kb 단위의파일크기를 출력<br>`-t` 파일을 최근에 생성된 시간순으로 출력<br>`-F` 파일의 형태와 함께 출력한다(`*` 실행파일, `@` 심볼릭링크, `/` 디렉토리)<br>`-R` 서브 디렉토리의 내용을 포함하여 출력(즉 하위 디렉토리 내용을 모두 출력)<br>`-S` 파일을 크기순으로 출력<br>`--help` 도움말 출력|
|`sudo`|관리자 권한으로 실행|`sudo su root` 루트로 로그인|
|`pwd`|전체 경로 확인||
|`find <옵션> <파일명>`|파일을 이름, 수정날짜, 소유주, 허가 등을 기준으로 하여 검색|`-name <파일명>` 파일명과 일치하는 파일을 찾기<br>`. -name <파일명>`현재 디렉토리에서 해당 파일 찾기<br>`-print` 현재의 경로 이름을 표시|
|`grep <옵션> <패턴> <파일명>`|파일 안 내용 찾기|`-c` 파일 내에서 패턴과 일치하는 줄의 총수를 구한다.<br>`-h` 여러 개의 파일을 검색할 때, 출력에 파일명이 나타나는 것을 방지한다.<br>`-i` 대, 소문자를 구별하지 않는다<br>`-n` 일치하는 줄의 줄 번호도 같이 출력한다<br>`-v` 패턴과 일치하지 않는 줄만 출력한다|
|`history`|히스토리 보기|종료->q<br>`history \ grep <검색어>` 히스토리 안 내용찾기|
|`clear`|화면 클리어 (window: `clr`)||
|`ipconfig`|ip 보기||
|`netstat -nat | grep TIME_WAIT | wc -l`|네트워크||
|`ps <옵션>`|현재 자신이 실행한 프로세스 목록을 출력||
|`lsof -i :<port번호>`|특정 포트 사용중인 프로세스 출력||
|`lsof -i -P | grep -i "listen"`|사용중인 프로세스 중 열린(listen) 포트 확인||
|`kill <-signalID> <PID>`|프로세스 kill|`-signalID` 프로세서에게 보낼 시그널을 지정한다. 시그널은 번호로 지정될 도 있고 시그널 이름으로 지정될 수도 있다.(`-9`)<br>`-l` 시그널 종류를 나열한다. 시그널의 종류는 시그널 번호 순서대로 이름으로 나열된다.|
|`jobs <옵션>`|현재 진행중인 작업들을 출력|`-l` 현재 작업하고 있는 디렉토리와 프로세스 그룹 ID를 출력|
|`mkdir <옵션> <디렉토리명>`|새로운 디렉토리 만들기 (make directory)|`-p` 만들고자하는 디렉토리의 상위 디렉토리가 없는 경우 상위 디렉토리까지 같이 생성|
|`rmdir <옵션> <디렉토리명>`|디렉토리 삭제 (remove directory)|`-p` 제거된 디렉토리에 대한 메시지가 표시됨<br>`-s` 제거된 디렉토리에 대한 메시지가 표시되지 않음(Default)|
|`touch <파일명.확장자>`|파일 생성||
|`rm <옵션> <파일명>`|파일 삭제, 삭제하고자 하는 디렉토리에 파일이나 부 디렉토리가 없어야만 삭제가 가능 (window: `del`)|`-f` 디렉토리 안의 파일을 삭제할 때 사용자에게 확인을 요구하지 않음<br>`-r` 인수 list 에서 지정한 디렉토리 혹은 그 아래의 subdirectory를 삭제<br>`-rf` 디렉토리 안에 파일, 디렉토리 등 모든 내용물들도 같이 삭제<br>`-i` whrite permission 이 없는 파일의 삭제를 위해 대화식으로 확인<br>`-p` 디렉토리 dir-name과 비어있는 부모 디렉토리를 사용자가 제거할 수 있으며, 전체 경로명이 삭제 되거나 어떤 이유로 인해 경로명의 일부가 남은것과 무관하게 표준출력에 메세지가 출력됨<br>`-s -p` 선택항목 지정시 표준오류에 출력되는 메시지를 삭제|
|`cp <옵션> <원본파일명> <목적파일명/디렉토리명>`|복사 (copy) (window: `copy`)|`-i` 파일 복사시 동일 파일명이 있을시에 사용자에게 덮어 쓸 것인지를 물어봄<br>`-f` 동일파일명 발생시에도 모두 강제적으로 복사함<br>`-p` 원본 파일의 시간 및 소유 권한 보존<br>`-r` 포함된 자식 디렉토리까지 모두 복사|
|`mv <옵션> <원본파일명> <새로운파일명/이동할디렉토리명>`|파일의 위치를 바꾸거나(경로) 이름을 바꾸는데(존재하지 않는 파일명) 사용<br>한번에 여러 개의 파일 이동도 가능(마지막만 이동할 디렉토리면됨)|`-f` 대상 파일이 존재할 대 새로운 파일로 대체<br>`-i` 대상파일이 있을 때 확인 후 작업, 같은 이름의 파일이 이미 존재하여서 그것을 덮어쓰기를 하지 않으려면 사용<br>`-f` 파일의 이동 과정을 보여준다.|
|`cat <파일명>`|텍스트 파일 열기||
|`vi <파일명>`|vim 편집기 실행||
|`ln -s <타겟 경로> <심볼릭 링크 경로>`|심볼릭 링크 걸기||
|`chmod <옵션> <파일명/디렉토리명>`|파일 권한 변경|`-R` 서브디렉토리까지 모든 파일의 권한 변경<br>`777` 모든 권리<br>`755` 읽기만<br>`700` 오너만 조작 가능|
|`chown <옵션> <파일명/디렉토리명>`|파일 소유자 변경|`-R` 서브디렉토리까지 모든 파일의 권한 변경|
|`zip <압축파일명.zip> <파일 또는 폴더>`|파일 또는 폴더를 zip으로 압축||
|`unzip <압축파일명.zip>`|압축 풀기||
|`diff`|두개의 파일이 일치되도록 변경시키기 위한 정보를 제공 man (differential file comparator)||
|`whoami`|현재 사용자||
