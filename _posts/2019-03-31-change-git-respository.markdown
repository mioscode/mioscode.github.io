#  기존 리포지토리 깔끔하게 pull / push
```
git pull
git add .
git commit -m "clean push"
git push
```
# 기존 리포지토리 remote 제거
```
git remote remove origin
```

# 새 리포지토리 remote 추가
```
git remote add origin https://github.com/playauto/리포지토리명 
```
